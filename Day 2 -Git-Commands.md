# Day 2 – Git Basic Commands

Today I learned basic Git commands used in DevOps.

Important Git Commands:

git init  
Initialize a new Git repository.

git clone  
Copy a repository from GitHub to local system.

git add  
Add files to staging area.

git commit  
Save changes with a message.

git push  
Upload local changes to GitHub repository.

git pull  
Download latest changes from GitHub.

Git is very important in DevOps because it helps teams track changes and collaborate on code.
